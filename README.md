# 📚 Library Management System

A full-stack **Library Management System** built with the MERN stack (MongoDB, Express.js, React.js, Node.js) and styled using **TailwindCSS**. This system allows librarians to manage books, users, and transactions, and enables users to browse, borrow, and return books.

## 🚀 Features

### 👩‍💼 Admin (Librarian)
- Login / Logout
- Add, edit, delete books
- View all issued books
- Manage users and their borrowing activity


### 👨‍🎓 User
- Register / Login
- Browse books by category, title, author
- Borrow and return books
- View borrowing history

## 🛠️ Tech Stack

| Layer       | Technology            |
|-------------|------------------------|
| Frontend    | React.js, TailwindCSS  |
| Backend     | Node.js, Express.js    |
| Database    | MongoDB                |
| Authentication | JWT (JSON Web Tokens) |
